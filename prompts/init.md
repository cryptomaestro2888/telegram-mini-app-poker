 Telegram mini-app that allows users to play poker with each other.

 ## Features

 - Create a room
 - Invite friends
 - Play poker
 - Chat

 ## Technologies

 - Telegram Mini App
 - Web App
 - Server
 - Telegram Bot API

 ## Database

 - MongoDB

 ## Server

 - Node.js
 - Express
 - Socket.io
 - Mongoose
 - JWT
 - Bcrypt
 - Joi
